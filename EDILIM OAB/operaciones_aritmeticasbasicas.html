La multiplicación es una operación matemática básica que se utiliza para sumar una cantidad repetidas veces. Es decir, es una forma rápida de agregar varias veces el mismo número.

La división es una operación matemática básica que se utiliza para repartir o distribuir una cantidad en partes iguales. Es una forma de compartir o dividir una cantidad en grupos equitativos.
